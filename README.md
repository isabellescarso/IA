# Plataforma RAG com Governança de Dados

Projeto desenvolvido para a disciplina de **Inteligência Artificial**.

## Sprint 1 — Definição do Produto

### Domínio
Saúde

### Empresa Fictícia
Laboratório de análises clínicas

### Problema de Negócio
Atualmente, muitas informações de saúde ficam distribuídas em diferentes sistemas e exames, dificultando a visualização geral do estado de saúde de uma pessoa.  
O problema que buscamos resolver é **como analisar e compreender de forma rápida como está a saúde do paciente a partir de seus dados clínicos**.

### Levantamento de Requisitos
- Centralizar informações médicas em um único sistema
- Permitir consultas inteligentes aos dados de saúde
- Utilizar inteligência artificial para auxiliar na interpretação das informações
- Facilitar o acesso e a compreensão do histórico de saúde

### Papéis Scrum
- **Product Owner:** responsável pela definição do produto e priorização do backlog  
- **Scrum Master:** responsável por garantir o processo ágil  
- **Time de Desenvolvimento:** responsáveis pela implementação da plataforma

### Entregável da Sprint
**Product Backlog inicial**

---

## Objetivo

Criar um sistema capaz de consultar dados corporativos e gerar respostas utilizando um **modelo de linguagem (LLM)** combinado com **busca semântica em banco vetorial**.

A plataforma utilizará o conceito de **Retrieval-Augmented Generation (RAG)** para permitir consultas inteligentes sobre dados estruturados e documentos.

## Tecnologias Utilizadas

- Modelos de Linguagem (LLM)
- Banco Vetorial
- Containers (Docker)
- Arquitetura de Dados Moderna
- MLOps
